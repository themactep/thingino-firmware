Motion Detection

This SoC/SDK has no IMP motion detection (IMP_IVS move API unavailable) — the feature is disabled.
Detection grid

Grid raster

The frame of the monitor stream is split evenly into these cells; each cell reports motion on its own. Watch it live on the Preview page (grid overlay button).

All settings apply immediately and persist. Actions on motion (email, Telegram, webhook, …) are configured under Tools → Send to services; timps triggers them via the motion.on_motion command in /etc/timps.conf (e.g. /usr/sbin/send2 -j, rate-limited by motion.cooldown_ms).
